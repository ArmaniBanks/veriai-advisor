# Verifiable AI Crypto Risk Advisor

A demo AI agent that analyzes crypto portfolio volatility using verifiable on-chain inference through the OpenGradient stack.

## Live Demo
https://veriai-advisor.vercel.app

## Overview
This project demonstrates how AI agents can generate transparent risk analysis using verifiable infrastructure.

Each query runs a structured pipeline:
- Portfolio exposure analysis
- Volatility model inference (ONNX)
- TEE-secured reasoning
- Risk recommendation output

## Tech Stack
- OpenGradient SDK
- ONNX
- ReAct agent pipeline
- Verifiable AI inference
